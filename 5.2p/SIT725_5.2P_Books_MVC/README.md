# SIT725 5.2P - Books MVC App

This is a simple Books catalog app built with Node.js, Express, and Vanilla HTML/CSS/JS using MVC structure.

## How to run

1. Open terminal in this folder
2. Install dependencies:
   npm install
3. Start the app:
   npm start
4. Open browser:
   http://localhost:3000

## Routes

- GET /api/books
- GET /api/books/:id
